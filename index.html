Language Identifier Project

Project Overview

This project identifies the language of an audio file using Google Cloud's Speech-to-Text API. The goal is to leverage cloud computing technologies to solve real-world problems like multilingual transcription.

Why I Chose This Project

With globalization, understanding multilingual content is increasingly important. This project combines cloud computing with machine learning to provide an innovative solution. It’s simple, practical, and showcases my ability to work with cutting-edge technology.

Features

  • Automatically detects the language of any audio file.
  • Supports multiple languages like English, Spanish, French, and more.
  • Provides transcription along with the detected language code.
  • Built using Google Cloud Platform’s Speech-to-Text API for scalability.

Technologies Used

  • Google Cloud Platform (GCP): Speech-to-Text API
  • Programming Language: Python
  • Cloud Storage: For hosting audio files
  • HTML/CSS: For creating this webpage
